# Quiz Application

A simple Java-based quiz application designed to help users test their knowledge on various subjects.

## Features

- Multiple-choice questions
- Immediate feedback on answers
- Score tracking

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- NetBeans IDE (optional but recommended)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/thebugbounter/Quiz-Application.git
   ```
2. Open the project in your preferred IDE (e.g., IntelIJ, NetBeans).
3. Build the project to resolve dependencies.

### Running the Application

1. Navigate to the `src` directory.
2. Run the `Main.java` file:
   ```sh
   java src/com/quizapplication/Main.java
   ```
